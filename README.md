# IoT 即時溫濕度監測系統

本專案展示了一個透過硬體或後端模擬產生溫濕度數據，存入 SQLite3 (`aiotdb.db`)，並由 Streamlit 即時讀取動態繪製「雙 Y 軸」圖表的完整流程。

## 📁 檔案說明
- `generate_data.py`：背景執行程式，每 2 秒產生一筆數值並持續寫入資料庫。
- `app.py`：前端 Streamlit 儀表板，即時查詢資料庫並繪製與更新圖表。

---

## 🛠️ 環境配置與安裝

本專案採用 `uv` 管理套件（專案配置位於 `pyproject.toml`）。

### 安裝專案套件
請在專案目錄下開啟終端機，並執行以下指令來自動建立虛擬環境並安裝所有依賴（包含 Streamlit、Pandas 與 Altair 等）：
```bash
uv sync
```

---

## 🚀 啟動與執行方式

為了讓系統展示動態更新的效果，您需要**開啟兩個獨立的終端機 (Terminal)**，針對這兩支程式分別獨立運行。

### ▶ 終端機 1：啟動資料生成器
負責將最新數據每隔 2 秒不斷送入 sqlite3 (如果想停止，可按下 `Ctrl+C`)
```bash
uv run generate_data.py
```

### ▶ 終端機 2：啟動即時儀表板
啟動後會自動在您的預設瀏覽器打開 `http://localhost:8501`。只要資料生成器正在運作，您就會在網頁上看到每兩秒自動更新的即時溫濕度圖表。
```bash
uv run streamlit run app.py
```
